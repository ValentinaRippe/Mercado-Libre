## Instrucciones de instalación

Clonar este repositorio. Se necesita `node` y `npm` instalada globalmente.  

Instalación:

`npm install`  

Ejecutar tests:  

`npm run test`  

Ejecutar para iniciar:

`npm run dev`  

Local:

`http://localhost:5173/`
